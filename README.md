
# Car Sales Power BI Dashboard

This repository contains an interactive Power BI dashboard developed to analyze and visualize car sales data. The dashboard helps in understanding overall business performance through clear, data-driven insights and well-structured visualizations.

## Project Overview
The Car Sales Dashboard focuses on providing a comprehensive view of sales performance across different dimensions such as time, region, and product. It enables users to identify trends, compare performance, and support data-backed decision-making.

## Key Features
- Overall sales and revenue key performance indicators  
- Monthly and yearly sales trend analysis  
- Top-selling car brands and models  
- Region-wise sales performance comparison  
- Interactive filters and slicers for detailed exploration  

## Tools and Technologies
- Power BI Desktop  
- Data modeling  
- DAX measures for calculated metrics  
- Interactive charts and visualizations
